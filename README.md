# Employee Payroll Management System

A simple C program to manage employee payroll records, including the ability to add employee details, view all employees, and generate individual pay slips. The program allows the user to input an hourly rate, which is used to calculate the employee's net salary based on hours worked.

## Features

- **Add Employee**: Add a new employee's details, including ID, name, hours worked, and hourly rate. The net salary is calculated and saved along with a timestamp.
- **View Employees**: Display a list of all employees with their ID, name, hours worked, hourly rate, net salary, and date added.
- **Generate Pay Slip**: Generate a pay slip for an individual employee by entering their employee ID. The pay slip is printed to the console and saved as a text file.
- **Data Storage**: Employee data is stored in a text file, `employee_data.txt`, for persistent storage across program runs.

## Installation

1. **Clone** this repository or copy the `payroll_system.c` file to your working directory:
   ```

   cd employee-payroll-system

2.**Compile** the program---->
  GCC payroll_system.c -o payroll_system
 
3.**Run** the program    ---->
